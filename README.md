# Samantha Nicole Olaco - Portfolio

## Dark Luxury Portfolio Website

A personal portfolio website built with React + Vite featuring a dark luxury theme with pink gradients and modern futuristic aesthetics.

## Tech Stack

- **React + Vite** - Modern React development
- **Framer Motion** - Smooth animations and interactions
- **Lucide React** - Consistent icon system
- **Fully Responsive Design** - Works on all devices

## Design Theme

- **Background:** Black/very dark theme
- **Accent Colors:** Pink gradients with soft neon glow
- **Style:** Modern, luxurious, futuristic digital assistant aesthetic

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:** Navigate to `http://localhost:3000`
